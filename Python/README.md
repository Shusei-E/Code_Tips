### String formatting
Example 1:
```python
print('Hello, %s!' % 'World')
```
Example 2:
```python
"My Name is {0}.".format("Kenta")
"My Name is {0}. Hello {1}".format("Ken", "Taro")
```
Example 3:
```python
"My Name is {person1}. Hello {person2}".format(person1="John", person2="Ken")
```
