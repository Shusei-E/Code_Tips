## Shortcut Keys
|            Function            |              Key             |
|:------------------------------:|:----------------------------:|
| 横に分割                       | Command + d                  |
| 縦に分割                       | Command + D                  |
| Windowを閉じる                 | Command + w                  |
| 分割Window間移動               | alt + Command + カーソルキー |
| 一つのウインドウを最大化／解除 | Command + shift + enter      |
| 新しいタブ                     | Command + t                  |
| タブ間移動                     | Command + カーソルキー       |
